# nova-stepper
Nova-stepper is a css library that helps to make a vertical stepper.

## Usage

```html
<ul class="nova-stepper">
  <li class="completed">Step 1</li>
  <li class="completed">Step 2</li>
  <li class="active">Step 3</li>
  <li>Step 4</li>
  <li>Step 5</li>
</ul>
```
